# IPM Logo Design — asset guide

Logo system for **Industrially Powered Magic**, built around the two mascots:

- **Sam** — the goblin artificer. She builds and enchants; she casts the electric-blue
  arcane current.
- **Shard** — Patrick's ratfolk monk, riding the wicker mechanical dragon he had
  enchanted (a graduate of *"So You Want to Be a Monk?"*).

The visual pun runs through both: Sam's **industrial magic** literally powers Shard's
flying construct. The whole system uses the site's existing brass-and-parchment palette
and the **Zilla Slab** wordmark font.

Open **`IPM Logo System.dc.html`** to see everything laid out in context.

## Wordmark

Live text, not baked into the art, so it stays sharp at any size:

- **Font:** Zilla Slab, 700 weight (matches the site).
- **Wordmark colour:** `#e7b45c` on dark / `#8a5a1e` on light.
- **Lightning accent:** `#7fd8ff` (arcane blue).

## Palette
`#0d0e10` near-black · `#c8963e` brass · `#6d5023` deep brass · `#3c2c11` outline ·
`#f6f0e4` parchment · `#7fd8ff` arcane blue.
